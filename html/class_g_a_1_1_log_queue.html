Detailed Description

ログを格納し、リスト化するクラス

シフト(perator<<)はstd::vectorを連結する。 define.hppで定義される(bool)GA::global::verbose_logがtrueであれば std::vectorを連結し、falseであればコードをキーとしてカウンタを合算する。
