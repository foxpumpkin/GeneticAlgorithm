Detailed Description

ログの実クラス

時刻(time_point)、メッセージ(string)、ID(uint64_t)、コード(uint64_t)、カウンタ(uint64_t)を持つ。 比較(operator==)はコードを比較し、和(operator+=)はカウンタを合算する。

ログレベルは下記URLを参考にしている。 https://qiita.com/nanasess/items/350e59b29cceb2f122b3
